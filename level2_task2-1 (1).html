About This Page

I built this responsive page as part of my Level 2 Task 2 submission for the Cognifyz Web Development Internship. The goal was to convert a basic webpage into a fully responsive design using CSS media queries and add a mobile-friendly hamburger navigation.

Try resizing the browser window — the two-column layout collapses into a single column on tablet, and the nav menu turns into a hamburger on mobile!

Techniques I Used

  • 📐
    CSS Grid — 2-col layout collapses at 900px breakpoint
  • 📱
    Hamburger Menu — Hidden on desktop, drawer opens on mobile
  • 🖋️
    Fluid Typography — clamp() scales headings smoothly
  • 🎨
    CSS Variables — Consistent theming across all breakpoints
  • 📏
    Media Queries — Breakpoints at 640px and 900px
